1. Become an IssueOwner

Sign up or log in to get access to the Issue Board. Each owner possesses a Participation Solvency metric, or PS, describing how active they are in the matter of solving other owners' solutions.

Each owner has a specific PS default that refreshes every month. This metric does not have a cap, but can turn negative if one owe too many Issue in a month. Creating Issue will cost you PS, but solving them will earn you more!

2. Create your Issue

Create your own issues after logging in at the Issue Board. In addition to Issue's name, description and deadline date, a particular Issue will also have UGC_Rating (Urgency rating) & DIF_Rating (Difficulty rating) that dictates its Participation Solvency cost when an IssueOwner attempt to create it.

This value is calculated by the following formula:

PS Cost = UGC_Rating * DIF_Rating


Each classification will have their own values, as per the following look-up table
UGC DIF Value
Misc Very Easy 1
Non-urgent Easy 2
Essential Medium 3
Urgent Hard 4
Very Hard Very Urgent 5

3. Solve Issue & the PS system

Without solving issues, owners will not regain any PS. The system incentivises harmonic distribution between declaring and solving Issue amongst the community, as well as "second-chances", which is why the value refreshes every month and only punish owners on owed amounts.

In a perfect world, each IssueOwner should have 100 PS or more by the end of month.
